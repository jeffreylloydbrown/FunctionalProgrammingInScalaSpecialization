  23. def upsweep(input: Array[Float], from: Int, end: Int, threshold: Int): Tree

    Traverses the part of the array starting at from going up to but not including end, and returns the reduction tree for that part of the array.

    Traverses the part of the array starting at from going up to but not including end, and returns the reduction tree for that part of the array.

    The reduction tree is a Leaf if the length of the specified part of the array is smaller or equal to threshold, and a Node otherwise. If the specified part of the array is longer than threshold, then the work is divided and done recursively in parallel.

    input

    The terrain data. All of it, not just a subarray.

    from

    The starting x position into input.

    end

    The ending position in input, exclusive. from <= x < end

    threshold

    Picks between sequential and parallel operations. end - from <= threshold runs sequentially. Otherwise split the segment roughly in half and run both in parallel.

    returns

    The intermediate tree of results representing each segment and its maximum angle tangent.
