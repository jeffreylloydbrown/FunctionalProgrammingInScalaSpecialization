class Boundaries extends AnyRef

Boundaries determine the size of the scene into which all bodies fit. It starts as a very large 2D Cartesian plane, with X increasing left to right and Y increasing top to bottom.

NOTE: We initialize backwards so we can always use math.min() to update the boundaries.min values and math.max to update the boundaries.max values.
